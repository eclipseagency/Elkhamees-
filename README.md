# مجوهرات الخميس · Al Khamees Jewellery ✨

الموقع الرسمي لعلامة **مجوهرات الخميس**، المتخصصة في المجوهرات الفاخرة والألماس المعتمد والتصاميم الأنيقة في المملكة العربية السعودية.

💎 ألماس معتمد · ✨ صياغة يدوية · 📍 المملكة العربية السعودية

## عن المشروع
واجهة متجر عربية RTL تضع المنتجات والتسوّق في الواجهة، مبنية كصفحة ثابتة سريعة وخفيفة:
- واجهة فاخرة بطابع العلامة (البني، العاجي، والذهبي) مع الشعار الأصلي.
- تصنيفات مرئية وشبكة منتجات قابلة للفلترة.
- مفضلة، سلة تسوق جانبية، وإرسال الطلب جاهزًا عبر واتساب.
- أقسام للمناسبات، الثقة، الشحن والتغليف وخدمة العملاء.
- متجاوب بالكامل مع سطح المكتب والجوال.

> الصور والأسعار الحالية تجريبية للعرض ويجب استبدالها بكتالوج المتجر المعتمد قبل الإطلاق التجاري.

## البنية
- `concepts/noir/index.html` — الصفحة الرئيسية المعتمدة حاليًا، ويحوّل `/` إليها عبر Vercel قبل تقديم الملفات الثابتة.
- `index.html` — النسخة السابقة محفوظة للرجوع والمقارنة، بدون حذفها.
- `assets/` — الشعار، الاسم بالخط العربي، وصور الألماس.

## التشغيل محلياً
```bash
python -m http.server 8000
# ثم افتح http://localhost:8000
```

## النشر
منشور على Vercel كموقع ثابت (Static). أي دفعة إلى الفرع `main` تُحدّث الموقع تلقائياً.

الاتجاه المعتمد هو تصميم **Noir** الداكن الذي وافق عليه العميل. تبقى الصور
والأسعار وبيانات التواصل خاضعة لاعتماد الكتالوج النهائي قبل الإطلاق التجاري.
